# Торговая система для GLD (Золото)

## Описание

Комплексная алгоритмическая торговая система для автоматического распознавания классических технических паттернов на рынке золота (GLD ETF) с мультитаймфреймовым анализом.

### Основные возможности:

✅ **Алгоритмическая детекция паттернов:**
- Голова и плечи (обычная и перевернутая)
- Двойное дно и двойная вершина
- Треугольники (восходящий, нисходящий, симметричный)

✅ **Мультитаймфреймовый анализ:**
- Одновременный анализ на 3 таймфреймах (1h, 4h, 1d)
- Подтверждение сигналов на нескольких ТФ
- Фильтрация ложных сигналов

✅ **Продвинутое управление рисками:**
- Фиксированный риск 2% на сделку
- Автоматический расчет размера позиции
- Stop-Loss и Take-Profit с соотношением 1:3

✅ **Полный бэктестинг:**
- 2 года исторических данных (май 2024 - май 2026)
- Учет спредов и проскальзывания
- Детальные метрики производительности

✅ **Профессиональные визуализации:**
- График роста капитала
- Примеры найденных паттернов
- Heatmap по месяцам
- Распределения P&L

## Использование отдельных модулей

### Загрузка данных

```python
from data_loader import DataLoader

loader = DataLoader(
    hourly_file='path/to/hourly_data.json',
    daily_file='path/to/daily_data.json'
)

df_1h, df_4h, df_1d = loader.load_data()
```

### Детекция паттернов

```python
from pattern_detector import PatternDetector

detector = PatternDetector(df_1d, timeframe='1d')
patterns = detector.detect_all_patterns()

for pattern in patterns:
    print(f"{pattern.pattern_type} - {pattern.direction}")
    print(f"Confidence: {pattern.confidence:.2f}")
    print(f"Key levels: {pattern.key_levels}")
```

### Бэктестинг

```python
from backtest_engine import BacktestEngine

engine = BacktestEngine(
    df_1h, df_4h, df_1d,
    initial_capital=100000,
    risk_per_trade=0.02
)

trades = engine.run_backtest()
metrics = engine.calculate_metrics()

print(f"Win Rate: {metrics['win_rate']*100:.1f}%")
print(f"Profit Factor: {metrics['profit_factor']:.2f}")
print(f"Total Return: {metrics['total_return_pct']:.2f}%")
```

## Логика детекции паттернов

### 1. Голова и плечи

**Обычная (медвежья):**
- 3 последовательных максимума
- Средний (голова) выше боковых (плечи)
- Симметрия плеч: разница ≤15%
- Линия шеи соединяет минимумы между пиками

**Перевернутая (бычья):**
- 3 последовательных минимума
- Средний (голова) ниже боковых (плечи)
- Аналогичные критерии симметрии

### 2. Двойная вершина/дно

**Двойная вершина (медвежья):**
- 2 примерно равных максимума (±3%)
- Минимум между вершинами = уровень поддержки
- Подтверждение: пробой поддержки вниз

**Двойное дно (бычье):**
- 2 примерно равных минимума (±3%)
- Максимум между днами = уровень сопротивления
- Подтверждение: пробой сопротивления вверх

### 3. Треугольники

**Восходящий (бычий):**
- Горизонтальное сопротивление (вариация <1%)
- Восходящая линия поддержки (положительный наклон)
- Сужение диапазона минимум на 30%

**Нисходящий (медвежий):**
- Горизонтальная поддержка (вариация <1%)
- Нисходящая линия сопротивления (отрицательный наклон)
- Сужение диапазона минимум на 30%

**Симметричный:**
- Обе линии сходятся
- Направление определяется при пробое

## Торговые правила

### Вход

1. Паттерн полностью сформирован
2. Пробой ключевого уровня с буфером:
   - Вверх: цена > уровень × 1.002
   - Вниз: цена < уровень × 0.998
3. MTF Score ≥ 2 (подтверждение минимум на 2 ТФ)
4. Нет открытых позиций

### Выход

- **Take-Profit:** Entry + 3 × Risk
- **Stop-Loss:** на противоположной стороне паттерна
- **Таймаут:** 30 дней с момента входа

### Размер позиции

```
Risk Amount = Capital × 0.02
Risk Per Share = |Entry - Stop|
Shares = Risk Amount / Risk Per Share
```

Ограничение: не более 95% капитала в одной позиции
